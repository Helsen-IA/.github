## Sobre

Software house especializada em **solucoes SaaS com inteligencia artificial**. Desenvolvemos plataformas completas — backend, frontend web e mobile — que resolvem problemas reais de gestao, vendas e atendimento para empresas no Brasil.

Combinamos **sistemas de gestao robustos** com **agentes de IA conversacionais**, entregando automacao de ponta a ponta via WhatsApp, dashboards analiticos e aplicativos mobile nativos.

## Arquitetura

Todos os projetos seguem uma arquitetura padronizada:

```
                    ┌──────────────────────────────┐
                    │           CLIENTES            │
                    │    Next.js  ·  Expo (Mobile)  │
                    └──────────────┬───────────────┘
                                   │
                          ┌────────▼────────┐
                          │   API Gateway   │
                          │ Fastify · REST  │
                          │   + WebSocket   │
                          └────────┬────────┘
                                   │
                    ┌──────────────┼──────────────┐
                    │              │              │
              ┌─────▼─────┐ ┌─────▼─────┐ ┌─────▼─────┐
              │ PostgreSQL │ │  Google   │ │  Stripe   │
              │  (Prisma)  │ │  Gemini   │ │ Payments  │
              └───────────┘ └───────────┘ └───────────┘
                                   │
                          ┌────────▼────────┐
                          │  WhatsApp Bot   │
                          │    (UAZAPI)     │
                          └─────────────────┘
```

**Padroes:**
- Monorepo com Turborepo — `apps/api`, `apps/web`, `apps/mobile`
- Multi-tenant com isolamento por organizacao
- Role-based access — Admin, Colaborador, Cliente
- IA integrada para automacao de processos e atendimento
